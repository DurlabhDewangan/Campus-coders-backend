APSGMNS Coders — Backend API

This repository contains the backend services for the APSGMNS Coders MVP — running from 10 Nov → 30 Nov as part of our launch phase.

Tech Stack
Node.js
Express
MongoDB + Mongoose
JWT Authentication
Cloudinary (Media Storage)
Multer (Uploads)

 Core Features (MVP)

✔ User Registration (Invite Code Required)
✔ Login + JWT Auth (Access & Refresh Tokens)
✔ Profile Setup & Avatar Support
✔ Follow / Unfollow
✔ Search Users
✔ Create / Delete Post
✔ Like / Unlike
✔ Comment System
✔ Infinite Scroll Feed
